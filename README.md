# Prueba
 Proyecto en el que se consume una API Rest y despliega la información.
 
 
 Instrucciones de ejecución local:
 *Tener instalado node js y react18*
 
 1.Abrir el CMD y ubicar el directorio en donde este guardada la carpeta del proyecto.
 2.Seleccionar el directorio con el comando CD
 3.Estando ya en el directorio del proyecto, ejecutar el comando npm start para iniciar el 
 servidor local.
